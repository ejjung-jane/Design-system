# Design System (React + TS)

React + TypeScript 기반의 개인 디자인 시스템 프로젝트입니다.  
Design Tokens(컬러/타이포/스페이싱) → UI 컴포넌트(Button/Input) → 문서화(Storybook) 순으로 확장합니다.

## Stack
- React + TypeScript (Vite)
- CSS Modules
- Design Tokens (CSS variables)
- Storybook (docs)

## Getting Started
```bash
npm install
npm run dev

## Tokens
- `src/styles/tokens/color.css`
- `src/styles/tokens/typography.css`
- `src/styles/tokens/spacing.css`

- Input: `src/components/Input
